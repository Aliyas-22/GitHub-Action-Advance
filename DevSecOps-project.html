✦ What I Learned Today
In DevOps we do Code → Test → Build → Deploy. In DevSecOps, we add a security check at every single stage.
SAST = Static Application Security Testing — it scans your code for security issues before the app even runs.
Practiced SAST tools — Bandit, Trivy, Hadolint & GitLeaks — each one scans a different layer for vulnerabilities.
If any security check fails, the pipeline stops. The app is denied — it never reaches production. 🚫
